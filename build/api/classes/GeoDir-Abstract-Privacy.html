in package
  • GeoDirectory

geodirectory/includes/abstracts/abstract-geodir-privacy.php : 19

Abstract class that is intended to be extended by specific privacy class.

It handles the display of the privacy message of the privacy id to the admin, privacy data to be exported and privacy data to be deleted.
