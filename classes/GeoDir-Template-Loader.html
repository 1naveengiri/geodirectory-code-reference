template_loader()

geodirectory/includes/class-geodir-template-loader.php : 88

Load a template.

public static template_loader(mixed $template) : string

Handles template usage so that we can use our own templates instead of the themes.

Templates are in the 'templates' folder. geodirectory looks for theme. overrides in /theme/geodirectory/ by default.

For beginners, it also looks for a geodirectory.php template first. If the user adds. this to the theme (containing a geodirectory() inside) this will be used for all. geodirectory templates.

$template : mixed
